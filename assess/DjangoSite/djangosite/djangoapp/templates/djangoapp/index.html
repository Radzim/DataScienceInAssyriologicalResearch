The proposed system is as follows: firstly, all the relevant available data about individual tablets would be parsed and connected into a single digital database. The database will have a UI, focused on allowing quick and computer-assisted data entry, that predicts the most probable input. We would also like to connect it to external data sources, to be overlaid on the interface to allow a more efficient workflow. On top of the data system, we would like to use machine learning to provide additional insights into the data, likelihood statistics of possible categorisation - for example attributing monarchs to partial dates provided in regnal years, and the detected relationships for each tablet, such as a graph of related tablets connected via recurring personal names (a basic example of this is attached). Please note that these plans will likely be narrowed down to fit the scope and timeline of the dissertation - we would love to hear your ideas on this.
